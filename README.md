# Valentine's Day App
An web application for spread ideas, and the important moments in your life with your patner.

## Installation

1. Clone the repository:
```
git clone https://github.com/JENEESHAPRIYADARSHANI/valentine-gift.git
```
2. Install the dependencies:
```
cd valentine-app
npm install
```

## Usage

1. Start the development server:
```
npm run dev
```
2. Open your web browser from the link that appear in the terminal.
